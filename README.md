# webplan项目README文档

基于 Web 的一站式个人效率管理系统，整合任务优先级管理、番茄钟专注计时与多维度数据统计看板，为个人用户提供简洁、高效的待办与专注管理解决方案。

---

## 项目简介

webplan 是一款基于 Flask 开发的轻量级个人效率工具，围绕「任务规划 \- 专注执行 \- 数据复盘」的效率闭环设计。系统采用 JWT 身份认证保障用户数据安全，基于 SQLite 实现开箱即用的轻量化部署，无需额外配置数据库，开箱即可搭建属于自己的个人效率管理平台。

---

## ✨ 核心功能特性

### 🔐 安全的用户身份认证体系

- 支持用户注册、登录、安全退出全流程操作

- 基于 JWT 实现身份认证，Token 存储于 HttpOnly Cookie 中，有效防范 XSS 安全风险

- 全局登录状态拦截，未登录用户自动跳转至登录页，实现多用户数据完全隔离

- 密码采用 Werkzeug 行业标准加密哈希存储，全程不泄露明文密码

### 📋 智能任务优先级管理

- 支持任务的新增、删除、完成状态一键切换，支持自定义任务描述

- 任务支持 3 级优先级设置（高 / 中 / 低），适配不同紧急程度的待办事项

- 任务列表智能排序规则：未完成任务优先展示、高优先级置顶、最新创建在前

- 任务与番茄钟深度联动，专注记录可直接绑定对应任务，实现目标与执行的闭环

### ⏱️ 番茄钟专注计时功能

- 内置标准番茄钟计时能力，支持自定义专注时长，适配不同专注习惯

- 专注记录支持两种模式：绑定对应待办任务、无绑定的自由专注模式

- 专注完成后自动保存记录，数据实时同步至统计看板，无需手动操作

- 专注页面自动加载用户未完成任务，一键选择绑定，操作流程极简

### 📊 多维度数据统计可视化看板

- **累计成就统计**：自动计算总专注次数、总专注时长、日均专注时长，直观展示长期积累

- **今日战况追踪**：实时更新当日专注次数、当日专注时长、累计完成任务数，掌握当日进度

- **专注分布可视化**：按任务维度统计专注时长占比，通过饼图直观展示精力分配情况

- **7 天专注趋势**：生成近 7 天每日专注时长趋势柱状图，清晰掌握效率变化规律

## 📁 项目结构

```Plain Text
webplan/
├── app.py                 # 项目主程序，包含路由、登录拦截装饰器、全量业务逻辑
├── config.py              # 项目配置文件，密钥、数据库连接等核心配置管理
├── models.py              # 数据库模型定义（用户、任务、番茄钟记录三大核心模型）
├── requirements.txt       # 项目依赖包清单
├── test.py                # 项目测试文件
├── templates/             # 前端模板文件夹（Jinja2 模板）
│   ├── index.html         # 系统主页/任务管理核心页面
│   ├── login.html         # 用户登录页面
│   ├── register.html      # 用户注册页面
│   ├── pomodoro.html      # 番茄钟专注计时页面
│   └── dashboard.html     # 数据统计可视化看板页面
├── static/                # 静态资源文件夹（存放CSS、JS、图片等前端资源）
└── app_v2.db              # 运行后自动生成的 SQLite 数据库文件
```

## 🚀 快速开始

### 环境要求

- Python 版本：3\.8 及以上

- 操作系统：Windows/macOS/Linux 全平台兼容

### 1\. 获取项目代码

```bash
# 克隆 GitHub 仓库到本地
git clone https://github.com/pycat13/webplan.git
# 进入项目根目录
cd webplan
```

### 2\. 安装项目依赖

```bash
pip install -r requirements.txt
```

### 3\. 启动项目服务

```bash
python app.py
```

启动成功后，终端会输出如下核心信息：

```Plain Text
* Serving Flask app 'app'
 * Debug mode: on
 * Running on http://127.0.0.1:5000
```

### 4\. 访问系统

打开浏览器，访问 `http://127\.0\.0\.1:5000` 即可进入系统。

> 首次启动会自动创建 SQLite 数据库文件 `app\_v2\.db`，无需手动执行数据库初始化命令，真正实现开箱即用。
>

## 📖 核心功能使用说明

1. **注册登录**：首次使用先完成账号注册，注册成功后自动跳转至登录页，登录后即可解锁全部功能

2. **任务管理**：在主页可新增任务，设置任务优先级，一键切换任务完成 / 未完成状态，删除不需要的任务

3. **番茄钟专注**：进入「番茄钟」页面，可选择关联对应任务（可选），点击开始即可进入专注计时，计时结束后自动保存专注记录

4. **数据复盘**：进入「数据看板」，可查看个人专注数据的多维度统计与可视化图表，全面掌握自己的效率情况与精力分配

## ⚙️ 生产环境部署提示

如果需要将项目部署到服务器正式使用，请务必完成以下安全与性能配置：

1. **修改核心密钥**：在 `config\.py` 中修改 `SECRET\_KEY` 为强随机字符串，禁止使用默认密钥

2. **关闭 Debug 模式**：`app\.py` 中启动服务时设置 `debug=False`，避免生产环境泄露敏感信息

3. **生产级服务部署**：推荐使用 Gunicorn 作为 WSGI 服务器，搭配 Nginx 反向代理部署

4. **数据库升级**：高并发或多用户场景，可将 SQLite 替换为 MySQL/PostgreSQL 等生产级数据库
